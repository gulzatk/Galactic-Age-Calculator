# Super Galactic Age calculator

#### Independent project for Epicodus, week-1, 01/11/2019

### By Gulzat Karimova

## Description

This is an application that user can calculate her/his age based on a planet's solar years. One Mercury year is 0.24 Earth year, a Venus year is 0.62, a Mars is 1.88 years and a Jupiter year is 11.86 Earth years.

## Specifications

1. The user input the age.
  * Example input: 27
  * Example output: "27"
2. The program returns the age in Mercury years.
  * Example input: 27
  * Example output: 27/0.24
3. The program returns the age in Venus years.
  * Example input: 27
  * Example output: 27/0.62
4. The program returns the age in Mars years.
  * Example input: 27
  * Example output: 27/0.24
5. The program returns the age in Jupiter years.
  * Example input: 27
  * Example output: 27/0.24
6. The program determines how many years a user has left to live on each planet.
  * Example input: 27
  * Example output: 78 / 0.24 - 113
7. The program return the number of years they have lived past the life expectancy on each planet.
  * Example input: 90
  * Example output: 90/0.24-78/0.24

## Setup/Installation Requirements

1. Clone this repository: https://github.com/gulzatk/Galactic-Age-Calculator.git
2. Open file in a terminal
3. Type $npm install to install all packages.
4. Run the program: $npm run start.
5. Test the program: $npm test.
